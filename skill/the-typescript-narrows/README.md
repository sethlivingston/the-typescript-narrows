# The TypeScript Narrows (Claude Skill)

Structured TypeScript opinions for AI-assisted development -- one opinion per decision point.

## Install

**Option 1: Copy into your project**

Copy the `skill/the-typescript-narrows/` directory into your project's `.claude/skills/` directory.

**Option 2: Reference from GitHub**

```
github:sethlivingston/the-typescript-narrows/skill/the-typescript-narrows
```

## What This Does

This skill provides 59 opinionated TypeScript guidelines covering type safety, error handling, async patterns, naming conventions, and more. Each opinion includes a stance, rationale, and do/don't code examples. See [SKILL.md](SKILL.md) for the full reference.

## Companion ESLint Plugin

For automated enforcement of 35 of these opinions, use the [`eslint-plugin-typescript-narrows`](https://www.npmjs.com/package/eslint-plugin-typescript-narrows) npm package.
