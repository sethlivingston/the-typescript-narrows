    - Import `basename` from `node:path`
    - Import `createRule` from `../utils/create-rule.js`
    - Import `minimatch` from `minimatch` (named import: `import { minimatch } from 'minimatch'`)
    - Define `type Options = [{ allowPatterns?: string[] }]`
    - Export named `banBarrelFiles`
    - Call `createRule<Options, 'banned'>({...})` with:
      - `name: 'ban-barrel-files'`
      - `meta.type: 'suggestion'`
      - `meta.docs: { description: 'Ban barrel files (index.ts re-export files)', opinionId: 'ban-barrel-files', recommended: true, requiresTypeChecking: false }`
      - `meta.messages: { banned: 'Do not use barrel files; import from source modules directly. (ban-barrel-files)' }`
      - `meta.schema`: One object element with `properties: { allowPatterns: { type: 'array', items: { type: 'string' }, description: 'Glob patterns for allowed barrel file paths (e.g., "src/index.ts")' } }, additionalProperties: false`
      - `defaultOptions: [{}]`
    - In `create(context, [options])`:
      - Define `INDEX_BASENAMES = new Set(['index.ts', 'index.js', 'index.mts', 'index.mjs'])`
      - Return visitor with `'Program:exit'(node)`:
        1. Get `filename = context.filename` and `base = basename(filename)`
        2. If `!INDEX_BASENAMES.has(base)` return early (not an index file)
        3. If `options.allowPatterns?.some(pattern => minimatch(filename, pattern))` return early (allowed)
        4. If `node.body.length === 0` return early (empty file is not a barrel)
        5. Check `isAllReExports = node.body.every(stmt => ...)`:
           - `stmt.type === 'ExportNamedDeclaration' && stmt.source != null` => true (re-export with source)
           - `stmt.type === 'ExportAllDeclaration'` => true (star re-export, always has source)
           - Otherwise => false
        6. If `isAllReExports`, report `{ node, messageId: 'banned' }`
